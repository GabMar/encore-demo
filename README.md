Symfony Webpack Encore Demo
========================

A simple Symfony application that can be used as a boilerplate.

What's inside?
--------------

This demo includes:

  * Symfony 3.3.9
  
  * Laravel Homestead 4.0.5
  
  * Webpack 3.7.1
  
  * Eslint 4.9.0
  
  * Vue.js 2.5.2
  
  * Bootstrap 4.0.0-beta
